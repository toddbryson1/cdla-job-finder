## Operator runbooks

### Bulk-publish remaining 411 URLs to Google Indexing API
```bash
cd ~/projects/cdla-job-finder
npx tsx scripts/bulk-publish-cycles.ts
```
- 180/day batch, resumes from `/tmp/cdla-indexing-progress.json`
- Re-run tomorrow, day after — ~3 more runs to finish 411 remaining
- Each run takes ~1-2 min; quota resets midnight Pacific

### TA Dedicated — pick mappings via review CLI
```bash
echo 'DATABASE_URL="<paste-from-vercel>"' > /tmp/cdla-prod.env
npx tsx scripts/ta-review.ts
```
- Walks every active opening
- Shows top-5 fuzzy candidates per opening with similarity scores
- Keys: `1-5` pick, `n` no match, `s` skip, `q` quit
- Mappings persist in `ta_opening_tab_mappings` table
- After review, re-run `npx tsx scripts/ta-sync.ts --apply` — minimal
  jobs upgrade to complete quality
